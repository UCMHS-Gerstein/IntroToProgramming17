# Turtle

* turtle graphics is a *module* we can import
* importing lets us use the functions built into turtle graphics
* This is part of the standard library, so there is official Python documentation

```python
import turtle

for n in range(12):
    turtle.forward(100)
    turtle.right(150)
```
